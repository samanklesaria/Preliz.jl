Preliz.jl

This is a Julia wrapper around the Python library preliz.

PreLiz is a tool for prior elicitation and visualization in Bayesian statistics.

About PreLiz

PreLiz provides tools for:

  • Prior elicitation
  • Prior visualization
  • Distribution exploration
  • Interactive prior selection

API Reference

Preliz.maxent — Method
maxent(
    _::Type{Distributions.Beta},
    lower,
    upper,
    mass
) -> Distributions.Beta{Float64}

Find the maximum entropy Beta distribution that with mass in the interval defined by the lower and upper end-points.

Examples

# Calculate the maxent Beta distribution with 90% of the mass between bounds
maxent(Distributions.Beta, 1.0, 8.0, 0.9)

Notes

This function uses the Python preliz package's maxent implementation internally.
